# simpleProject1
게시판 &amp; 뉴스  검색

-------------------------------------------------------------------------------------------------

* 언어 : 파이썬 ( python version = 3.8 )
* 프레임워크 : Django ( django version = 2.2 )
* 가상환경 = pipenv 이용.

------------------------------- ------------------------------------------------------------------
 ` 기능 `
* 게시판 내 검색 기능 
* oepn-api를 통한 크롤링 방법을 이용하여 외부 영문 뉴스 검색 기능 구현  (( 암호화폐 또는 블록체인 관련 검색만 가능합니다.*))
